ZetaPush _built-in cloud services_ are available in `@zetapush/platform-legacy` module.
Add this module to your `package.json` by running the following command:

.Install @zetapush/platform-legacy using npm
[source, console, linenums]
----
$ npm install --save @zetapush/platform-legacy
----
