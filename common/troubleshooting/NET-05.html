Behind proxy

If Node.js/npm reports one of theses errors:

  • ENETUNREACH

  • ECONNREFUSED

It may mean that your system or Node.js/npm is configured with proxy settings. Please check that your proxy configuration is correct.

Proxy configured system-wide

System-wide proxy

Linux

In your terminal, execute the following commands to know if your system uses a proxy:

echo $http_proxy
echo $HTTP_PROXY
echo $https_proxy
echo $HTTPS_PROXY

Proxy configured for npm

In your terminal, execute the following commands to know if npm uses a proxy:

npm config get proxy
npm config get https-proxy

If at least one command displays any other result thant null, then npm is configured to use this proxy. If you want to disable npm proxy, you can execute:

npm config rm proxy
npm config rm https-proxy
